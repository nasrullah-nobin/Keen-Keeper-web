# 🌿 KeenKeeper

## 📌 Project Overview

**KeenKeeper** is a modern relationship management web app that helps users track, organize, and reflect on their personal connections. It allows users to log interactions like calls, texts, and video chats, visualize relationship data, and maintain meaningful connections over time.

---

## ✨ Description

KeenKeeper acts like a **personal relationship dashboard**. Users can view their interaction timeline, filter activities, and analyze communication patterns through visual charts. The goal is to make relationships more intentional and meaningful through simple tracking and insights.

---

## ⚙️ Technologies Used

* **Next.js** (App Router)
* JavaScript
* **Tailwind CSS + DaisyUI**
* **Recharts** (for data visualization)
* **Context API** (state management)

---

## 🚀 Key Features

### 1. 📊 Interactive Timeline

* Track all interactions (Call, Text, Video)
* Clean and readable UI
* Real-time filtering system

### 2. 🔍 Smart Filtering System

* Filter timeline by interaction type
* Dynamic UI update without page reload
* Smooth user experience

### 3. 📈 Analytics Dashboard

* Pie chart visualization of interaction types
* Responsive and mobile-friendly chart
* Helps users understand communication patterns
